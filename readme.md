# WordPress Assessment Submission

This repository contains my custom WordPress theme created for the technical assessment.

## Stack

* WordPress
* Advanced Custom Fields (ACF)
* Custom Post Types
* Custom CSS / JavaScript

## Setup

1. Copy the theme folder into `wp-content/themes/`
2. Install and activate ACF
3. Activate the theme from WordPress admin

## Notes

* Homepage has been implemented based on the provided design
* Responsive behavior is partially implemented and requires further refinement
* Minor visual differences may remain
